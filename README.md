<h2> Introduction </h2>
A Todo web app made with <a href="djangoproject.com"> Django </a> web framework.

<h3> Requirements </h3>
<a href="python.org"> Python </a>

Django
```
pip install django
```

<h3> Cloning the Repository </h3>
Clone the repository using

```
git clone https://github.com/Klvxn/Todo-App.git
```
